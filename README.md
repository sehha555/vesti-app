### OpenWeatherMap API Key

The weather service requires an API key from OpenWeatherMap.

1.  **Sign up** for a free account on [OpenWeatherMap](https://openweathermap.com/appid).
2.  **Find your API key** on the [API keys page](https://home.openweathermap.org/api_keys).
3.  **Copy the key** and paste it into your `.env` file as `WEATHER_API_KEY`.

```
WEATHER_API_KEY=your_api_key_here
```
