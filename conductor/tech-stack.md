# 技術棧

根據對現有程式碼和專案文件的分析，以下技術構成了本專案的核心技術棧：

## 1. 核心技術
-   **程式語言**：TypeScript
-   **前端框架**：Next.js (使用 App Router)
-   **後端框架**：用於網路應用程式介面的 Next.js API Routes，輔以位於 `services/` 目錄中的專用服務，用於後端業務邏輯。此架構是單一儲存庫（monorepo）設置的典型特徵。
-   **資料庫**：Supabase，利用 PostgreSQL 提供其關係型資料庫功能和整合服務。

## 2. 關鍵整合與工具
-   **影像管理**：Cloudinary 用於強大的影像儲存、優化和交付，以及 remove.bg 用於自動背景移除。
-   **天氣資料**：OpenWeatherMap API 用於獲取準確即時的天氣資訊，以支持每日穿搭推薦。
-   **單一儲存庫工具**：採用 Turborepo 管理單一儲存庫結構，促進專案內不同應用程式和套件之間的有效建構、測試和部署。
